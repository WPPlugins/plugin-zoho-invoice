== Description ==
Plugin criado com a função de sincronizar os clientes do Zoho Invoice com os Usuários do WordPress.

A - O que eu preciso fazer antes de utilizar este Plugin?

Para começar a usar este plugin faça o seguinte.
1. Cadastrar no site do Zoho "https://invoice.zoho.com/signup" para obter um usuário e senha, caso já tenha se cadastrado pule esta parte.
2. Um servidor seu ou de algum amigo para que possa hospedar o wordpress e instalar este plugin.
3. Uma conexão com a internet obviamente.
4. Ativar sua API, caso não saiba leia a etapa B.
5. Obter sua chave APIKEY do Zoho Invoice, caso não saiba leia a etapa C.
6. Cadastrar a APIKEY nas configurações aqui no WordPress

B - Como faço para ativar API no site do Zoho Invoice após acessar minha conta?

1. Acessar com o usuário e senha no Zoho Invoice
2. Clicar na guia Configurações.
3. Clicar no link Configurações API
4. Clicar em "Sim, eu quero habilitar o acesso API".
5. Clique em Salvar.

C - Como faço para obter uma chave de API?

Você pode solicitar sua chave de API, "http://zapi.zoho.com/apigen.do".
Após você solicitar esta chave o Zoho Invoice encaminhará a APIKEY para o seu email.


***** O Plugin somente importa os clientes do Zoho com "EMAIL" cadastrado ******

Plugin created with the function of synchronizing the customers of Zoho Invoice with WordPress users.

A - I need to do before using this plugin?

To start using this plugin do the following.
1. Register on the site of Zoho "https: / / invoice.zoho.com / signup" to get a username and password, if already registered skip this part.
2. Its a server or a friend so you can host your wordpress and this plugin.
3. A connection to the internet obviously.
4. Activate your API, if you are unsure please step B.
5. Get your key apiKey of Zoho Invoice, if you are unsure please step C.
6. Register here to apiKey settings in WordPress

B - How do I activate API in Zoho Invoice site after accessing my account?

1. Login with your username and password in Zoho Invoice
2. Click the Settings tab.
3. Click the Settings link API
4. Click "Yes, I want to enable access API."
5. Click Save.

C - How do I get an API key?

You can request your API key, "http://zapi.zoho.com/apigen.do.
After you apply this key to transmit Zoho Invoice apiKey to your inbox.


***** The plugin imports only customers with Zoho "EMAIL" registered ******

== Installation ==

1. Faça upload dos arquivos para a pasta `/wp-content/plugins/`  (mantenha a pasta original do plugin)
2. Ative o plugin na interface de 'Plugins' do WordPress
3. Na Aba de opções configure o plugin
